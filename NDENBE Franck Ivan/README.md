## Description des TP

### TP5 — Parseur de fichier

Lecture et parsing d'un fichier texte avec gestion d'erreurs.

Concepts abordés : parsing de texte, `Result` et propagation d'erreurs, utilisation de `thiserror`.

### TP6 — Abstraction générique

Implémentation d'une abstraction I/O via des traits.

Concepts abordés : traits, polymorphisme statique et dynamique, `Box<dyn Trait>`.

## Exécution des TP

```bash
cd tpX-nom_du_tp
cargo run
```

## Utilisation du Makefile

```bash
make run-tp5
make run-tp6
make build-all
make clean-all
```
