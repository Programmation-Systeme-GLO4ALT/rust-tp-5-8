# TP1 - Calculatrice CLI

Ce TP correspond à :
- installation et vérification de l'environnement,
- création d'un projet Cargo,
- implémentation d'une calculatrice CLI,
- bonus mode interactif. fileciteturn2file0

## Lancer
```powershell
cd "D:\EXAMEN RUST\tp1_calculatrice_cli"
cargo run -- 12 + 5
cargo run -- 10 / 2
cargo run
```

## Comportement
- `cargo run -- 12 + 5` : mode arguments
- `cargo run` : mode interactif
- `quitter` : sortie du mode interactif
