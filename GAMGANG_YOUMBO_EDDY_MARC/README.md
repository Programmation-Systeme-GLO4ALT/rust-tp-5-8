# TP Rust fusionnes

Ce depot regroupe les 4 TP Rust de l'examen dans une seule arborescence.

## Structure
- `TP1/calculatrice_cli`
- `TP2/gestion_taches_ownership`
- `TP3/analyseur_chaines`
- `TP4/gestionnaire_processus_os`

## Contenu
Chaque projet conserve ses sources Rust, son `Cargo.toml`, son `Cargo.lock` si present, et sa documentation utile.
Les dossiers de build (`target/`), les dossiers IDE (`.vs/`) et les archives ne sont pas conserves.
